ReadSettings For Python

Easily create, edit and remove a customized settings file which you can use for storing all of the settings for your application. At the moment, ReadSettings is compatible with Python 3 but support to more programming langauages may come in the future.

Version 1.0 Documentation

Installation

You can install ReadSetting for Python with pip.

pip install readsettings

Usage

Importing

Simply include the following line at the top of your Python file.

import readsettings

Create or reset a settings file

This function creates or resets a settings file

readsettings.fresh_file(settings_path)

Create or change of a setting

This function creates or changes a setting.

readsettings.set_value(settings_path, setting_name, value_content)

Rename a setting

This function renames a setting.

readsettings.rename_setting(settings_path, setting_name, new_setting_name)

Get the value of a setting

This function returns the value of a setting.

readsettings.get_value(settings_path, setting_name)

Get the raw JSON

This function returns the raw JSON of the settings file.

readsettings.raw_content(settings_path)
